# FormularioTecnico

Aplicación web para técnicos de campo de **WIN Perú** (ISP). Permite registrar instalaciones de fibra óptica: topología de red, mediciones de cobertura Wi-Fi, decodificadores WINBOX y Smart TVs.

## 🚀 Inicio Rápido

```bash
npm install
npm run dev
```
